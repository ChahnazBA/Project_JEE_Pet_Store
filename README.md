# 🐶 Pet Adoption Management System

A web-based platform built with **Java EE** that connects pet sellers with potential adopters.

### 🚀 Key Features
- **Dynamic Catalogue:** Buyers can browse available pets with real-time availability filters.
- **Adoption Workflow:** A 3-step transaction system:
  1. **Available:** Open for requests.
  2. **Pending:** Request received and animal hidden from catalogue.
  3. **Sold:** Transaction finalized with an automated invoice.
- **Dual Dashboards:** Custom interfaces for sellers to manage their listings and for buyers to track their requests.

### 🛠️ Tech Stack
- **Backend:** Java (Servlets, JavaBeans)
- **Frontend:** JSP, JSTL, CSS3
- **Architecture:** MVC (Model-View-Controller)
- **Design Pattern:** Singleton (Data Management)
